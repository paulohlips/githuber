## Getting started

Clone repo:

```
$ git clone
```

Inside of projects folder, install all dependencies:

```
$ yarn (or npm install)
```

To run on virtual device or on a mobile:

```
$ react-native run-android
```

or

```
$ react-native run-ios
```
